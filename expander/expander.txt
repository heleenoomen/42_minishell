
		DOLLARSIGN

		1. NO QUOTES

			a. EXISTING VARIABLE
			$HOME							/Users/hoomen		interpretation

			b. NON EXISTING VARIABLE
			$foo							empty string		yields empty string

			c. EXISTING VAR + OTHER TXT
			$HOMEbla						empty string		yields empty string

			d. OTHER TXT + $ + EXISTING VAR
			bla$HOME						bla/Users/hoomen	interpretation

			e. OTHER TXT + $ + EXISTING VAR + OTER TXT
			aaaa$HOMEbbbb					aaaa				token everything from $ on removed

			f. ON ITS OWN
			$								$					literal

			g. DOUBLE
			$$								63740				yields current PID

			h. TRIPLE
			$$$								63740$				yields current PID + literal

			i. FOLLOWED BY NUMBERS
			$0								bash				terminal info
			$1													yields empty string

			j. COMBI WITH NUMBERS
			aaaa$0							aaaabash			interpretation
			aaaa$0bbbb						aaaabashbbbb		interpretation
			
			k. COMBI WITH EMPTY STRING
			$""								empty string		yields empty string



	2. SINGLE QUOTES
			
			a.inside single quotes	
			'aaaa$HOMEaaaa'					aaaa$HOMEaaaa		ALWAYS LITERAL

			b.variable in single quotes
			$'HOME'							HOME				literal, dollar sign and quotes removed

			c.var in single quotes, followed by other txt
			$'HOME'aaa						HOMEaaa				literal, dollar sign and quotes removed
	
			d.var is number, in singe quotes
			$'0'							0					literal, dollar sign and quotes removed

	
	3. DOUBLE QUOTES

			a. INSIDE DOUBLE QUOTES
			"$HOME"							/Users/hoomen		interpreted

			b. INSIDE DOUBLE QUOTES, FOLLOWED BY OTHER TEXT
			"$HOME"aaa						/Users/hoomenaaa	interpreted

			c. INSIDE DOUBLE QUTES, OTHER TEXT BEFORE AND AFTER
			aaa"$HOME"aaa					aaa/Users/hoomenaaa	interpreted

			b. INSIDE DOUBLE QUOTES, NON EXISTING VARIABLE
			"$FOO"							empty string		yields empty string

			c. INSIDE DOUBLE QUOTES, NON EXISTING VAR, TXT BEFORE AND AFTER
			aaa"$FOO"bbb					aaabbb				everything in quotes omitted

			d. DOUBLE QUOTES, SINGLE QUOTES WITHIN
			"$HOME'aaa'"					/Users/hoomen'aaa'	interpreted, single quotes preserved
			"'a'$HOME'b'"					i'a'/Users/hoomen'b'	idem

			e. ONLY VARIABLE IN DQ
			$"HOME"							HOME				literal, quotes and dollar sign removed
			
			f. DOUBLE QUOTES, VAR IN SINGLE QUOTES
			"$'HOME'"						$'HOME'				literal
			
			f2. DOUBLE QUOTE, VAR AND SIGN IN SINGLE QUOTES
			"'$HOME'"						'/Users/hoomen'		interpre

			g. DOUBLE QUOTES, ON ITS OWN
			"$"								$					literal

			h. DOUBLE QUOTES, WITH NUMBER
			"$0"							terminal info		interpreted
			"$1"							empty string		yields empty string

			j. NON EXISTING VAR, TXT BEFORE AND AFTER
			aaa"$foo"bbb					aaabbb				deletion
